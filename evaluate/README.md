
## Installation

To use this code, you need to have the following dependencies installed:

- python 3.11
- torch 2.5.1
- cuda 12.1+

## Usage

1. Clone this repository.
git clone https://github.com/MaigeWhite/lighteval.git
2. pip install -e lighteval/

```bash
# default eval math_500
bash eval_rwkv.sh

```
